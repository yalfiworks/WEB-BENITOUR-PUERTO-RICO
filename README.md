# Benitour Puerto Rico

Next.js landing page for Benitour Puerto Rico.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## SEO

- Canonical URL: `https://benitourpuertorico.com`
- Sitemap: `/sitemap.xml`
- Robots: `/robots.txt`
- Favicon assets: `/favicon.ico`, `/favicon-192.png`, `/apple-touch-icon.png`
- Structured data: Organization + TouristTrip JSON-LD
